Overview
========

.. warning::
   This is an experimental release. Documentation remains a work in progress. Some sections may be incomplete or under development.


access-cryosphere-data-pool is a Python library designed to facilitate access to cryosphere-related datasets. It provides tools for discovering and accessing data related to glaciers, ice sheets, and sea ice on `NCI Gadi <https://opus.nci.org.au/spaces/Help/pages/90308778/0.+Welcome+to+Gadi#id-0.WelcometoGadi-Overview>`_.
